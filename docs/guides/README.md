# Base user guides

These guides explain every part of Base (base.integratedai.com.au) in plain
language. The audience is internal: Marnie and the IAI agents. They describe
what the shipped code actually does, nothing aspirational.

Rules for this folder:

- Accurate to the deployed product. When a behaviour changes (a page, a
  limit, a flow), the matching guide changes in the same PR. A stale guide is
  a defect.
- Plain language, Australian English, no em dashes.
